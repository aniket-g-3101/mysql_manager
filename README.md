# MySQL Manager

MySQL Manager is a Django-based web application that allows users to manage MySQL databases using a simple web interface instead of the command line.  
The project focuses on backend development, database handling, and secure user interaction.

This application was developed as a final-year academic project.

---

## Project Description

The goal of this project is to provide an easy-to-use interface for managing MySQL databases.  
Users can create and view databases, access tables, and perform basic database-related operations through a browser.

The project demonstrates practical use of Django with MySQL in a real-world scenario.

---

## Key Features

- User authentication and session handling
- Create and manage MySQL databases
- View database tables and records
- Perform basic database operations via UI
- Secure Django backend
- Simple and responsive interface

## Running the Project

To run the project locally, the following steps are followed:

- Clone the repository
- Create and activate a Python virtual environment
- Install required Python packages
- Configure MySQL credentials in Django settings
- Apply database migrations
- Start the Django development server

The application runs on:
http://127.0.0.1:8000/
